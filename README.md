# RIME-YANAMI
Rime全拼自用配置方案。



### 使用

- 拆字辅码 - \`h
- 大写数字 - \`r
- unicode - \`u
- 计算器 - \`c
- 大写日期 - \`n
- 当前日期 - rq
- 时间 - sj
- 星期 - xq
- ISO 8601 - dt
- 时间戳 - ts
- 当前农历 - nl
- 拆字反查 - ;
<!-- - 日语输入 - \`j
- 英语输入 - \`e
- 选中词组的第一个字 - [
- 选中词组的最后一个字 - ] -->
